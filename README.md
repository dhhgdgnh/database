## 适用于OpenClash分流规则和ICO图标


## 📦 目录结构

```
.
├── README.md              ← 项目说明（这个文件）
└── clash            
    ├── confg/                 ← 各位大佬的源文件
    ├── icon/                  ← 是本仓库制作的图标文件 
    ├── rul├e/                 ← 是本仓库私人规则，可按需引用
    ├──Clash_config.yaml       ← 是本仓库更改的 Clash 配置样例
    └── …
```






## 🚀 快速开始（Start.yaml）

仓库中的 `Clash_config.yaml` 是一个完整的 Clash 配置样例，包含：


* 多种策略组
* 常见规则调用

你可以将其下载，直接在你的 Clash 客户端中加载使用。  
下载配置文件修改proxy-providers下的url: "订阅地址"为自己的订阅链接导入客户端使用即可。
